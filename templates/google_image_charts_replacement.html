    Getting Started

    Welcome to the QuickChart manual for replacing Google Image Charts with QuickChart.

    QuickChart is an open-source service that renders static chart images. You can host your own server using the source code or a Docker container, or you can use the reliable QuickChart.io web service and not worry about hosting.

    Most QuickChart users prefer the open-source Chart.js format over Google Image Charts format due to its superior flexibility, strong community, and open-source nature. However, you may have already built your application on top of Google Image Charts. This guide is dedicated to making the switch as easy as possible.

    Migrating from Google Image Charts

    Migrating is easy. Just replace chart.googleapis.com with quickchart.io in the URL.

    For example, the URL: https://chart.googleapis.com/chart?cht=bvg&chs=300x200&chd=t:5,5,5|10,10,10|15,15,15&chco=4d89f9,c6d9fd,00B88A&chds=0,20&chbh=a&chxs=0,000000,0,0,_&chxt=y&chm=N,000000,0,,10|N,000000,1,,10|N,000000,2,,10

    Changes to: https://quickchart.io/chart?cht=bvg&chs=300x200&chd=t:5,5,5|10,10,10|15,15,15&chco=4d89f9,c6d9fd,00B88A&chds=0,20&chbh=a&chxs=0,000000,0,0,_&chxt=y&chm=N,000000,0,,10|N,000000,1,,10|N,000000,2,,10
